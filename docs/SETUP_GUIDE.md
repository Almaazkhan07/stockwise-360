# StockWise 360 Setup Guide

This folder contains the implemented project described by the original setup guide:

- `backend/` is the Spring Boot 3 API.
- `frontend/` is the React 18 + Tailwind user interface.
- `docs/database-schema.sql` contains the MySQL bootstrap commands.

Follow the root `README.md` to create the database, run the backend, and run the frontend.
